# 엘리엇 파동 — 리서치 결론과 재설계 방향

## 왜 지금까지 실패했나 (막대 긋기)
현 구현은 "고립된 5파를 슬라이딩으로 찾기" 휴리스틱. 그래서 (1)연속 교대 없음 (2)진짜 프랙탈 재귀 없음 (3)파동 비율/투영 디테일 없음. 사용자가 원하는 건 **전 구간 5파+3파 중단없는 교대 + 각 파동 내 재귀 + 파동 크기·기간 투영**.

## 리서치 핵심 (권위 소스 교차검증)

### 1. 재귀 문법 (formal grammar)
```
Cycle       := Motive Corrective   (반복: M C M C …)
Motive      := W1 W2 W3 W4 W5       (1,3,5=Motive 재귀 5분할 · 2,4=Corrective 재귀 3분할)
Corrective  := Zigzag(5-3-5) | Flat(3-3-5) | Triangle(3-3-3-3-3) | Combination(WXY 7·11스윙)
Base case   := 스윙이 너무 작아(피벗<~5) 더 못 나누면 leaf(monowave)
```
완전 사이클 = 8파(5+3), 한 등급 아래 = 89+55파(피보나치, 프랙탈 자기유사).

**3대 하드룰(불변식, 위반 시 무효)**: R1 2파는 1파 시작 비침범 · R2 3파는 1/3/5 중 최단 아님 & 1파 끝 돌파 · R3 4파는 1파 영역 비침범(대각 예외).
**가이드라인(소프트, 순위용)**: 교대(2파 급하면 4파 완만)·연장(1/3/5 중 하나, 보통 3파)·동등(3파 연장 시 5파≈1파).

### 2. 연속·다등급 라벨링 알고리즘 (3조각 조합 — 단일 오픈소스는 없음)
1. **재귀 ZigZag 피라미드**: 기본 ZigZag → 임계를 배수로 키워 상위 등급 피벗. **상위 피벗은 하위의 부분집합**(nesting 보장) → gapless 재귀 가능.
2. **최상위 등급 gapless 체인**: 끝피벗=다음 시작피벗으로 연결(alessioricco `buildWaveChainSet`), 하드룰 검증. → 전 구간 M→C→M 무중단 체인.
3. **각 파동 재귀**: 하위 등급 피벗으로 부모(motive=5·corrective=3) 재파싱. 등급 2~3단계에서 멈춤(더 깊으면 노이즈).
4. **모호성**: 유효 라벨이 여럿 공존(엘리엇 원저자도 인정) → primary + alternate 1개 + 신뢰도.

### 3. 가격 투영 (고신뢰, python-taew 코드와 문서 일치)
| 투영 대상 | 입력 | 공식 | 배수 |
|---|---|---|---|
| 2파 끝 | W1 | W1끝 − len(W1)×r | 0.5·0.618(주)·0.786 |
| 3파 끝 | W1,W2 | W2끝 + len(W1)×r | **1.618(주)**·2.618·4.236 |
| 4파 끝 | W3 | W3끝 − len(W3)×r | **0.382(주)**·0.236 |
| 5파 끝 | W1,W3,W4 | W4끝 + len(W1)(동등) 또는 +(len1+len3)×{0.382,0.618} | — |
| C(지그재그) | A,B | B끝 + len(A)×r | 0.618·1.0·1.618 |

투영은 **단일 선이 아니라 구간(zone)**으로 표시.

### 4. 시간 투영 (저신뢰 — 문서 근거 약함, 실험적)
`duration(참조파) × 같은 피보배수`. 예: 3파 시간 ≈ 1파 시간 ×1.618. 피보 시간대(1,2,3,5,8,13봉). **회색·토글·실험 표기 필수.**

## 실측으로 확인한 현실 (정직성)
유니버스 실측 결과 **깨끗한 gapless 5-3 교대·클린 프랙탈은 교과서 이상형**이고 실데이터는 저항한다:
- 재귀 nesting 보장돼도 major 다리 내부 minor 스윙이 3·5·7·11·21개로 불규칙(복합조정).
- 5레그를 motive 로 강제하면 하드룰을 거의 못 통과(연속 파싱 시 대부분 '미완').
- 우측 끝(진행 파동)은 본질적으로 확정 불가.

→ 결론: **연속성은 "구조적 위상 교대"로 보장하되(전 구간 gapless), 각 파동의 '진짜 5파 여부'는 하드룰/피보 신뢰도로 차등 표시**하는 게 현실적이고 정직하다.

## 재설계 안 (제안)
- **A. 재귀 ZigZag 등급 피라미드**(2~3등급, nesting 보장)
- **B. 각 등급에서 gapless 위상 교대**: 피벗을 좌→우로 motive(≤5레그)/corrective(≤3레그) 번갈아 배정 → 전 구간 중단없이 라벨. 규칙 통과분은 고신뢰(실선·1~5 라벨), 미달분은 저신뢰(옅게·물음표).
- **C. 프랙탈**: 상위 파동 안에 하위 등급 위상 교대를 재귀 표시.
- **D. 가격 투영 zone**: 최근 완성 파동들로 다음 파동 목표 구간(피보) 계산·표시.
- **E. 시간 투영**: 실험적·회색 토글.
- **F. 신뢰도·대안**: primary + alternate, 우측 끝은 provisional.
